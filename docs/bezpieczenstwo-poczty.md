# Bezpieczeństwo poczty elektronicznej
Obecnie w Polsce dominują oszustwa komputerowe (raport CERT Polska 2025), a poczta elektroniczna to jedno z najczęstszych miejsc ataków na pracowników.

##Rozpoznawanie zagrożeń
##Jakie środki ostrożności należy zachować?
1. Nie należy używać prywatnych kont czy urządzeń w celach służbowych.
2. Nie należy używać służbowych urządzeń, kont w celach prywatnych, nie należy też umożliwiać dodstępu do nich osobom trzecim, nawet rodzinie.
3. Zawsze należy sprawdzać czy domena jest prawidłoPrzykłady - jak może wyglądać atakwa.
4. Nie należy podoawać nikomu swojego hasła
5.Stosować trudne so skopiowania, czy odgadnięcia hasła:
- warto zwrócić uwagę na odpowiednią długość oraz wykorzystanie znaków specjalnych.
- należy unikać powiązań z imieniem, nazwiskiem, datą urodzenia, peselem i z innymi hasłami które już posiadamy.
6. Warto włączyć uwierzytelnianie dwuskładnikowe
7. Należy pamiętać o posiadaniu aktualnego programu antywirusowego.
